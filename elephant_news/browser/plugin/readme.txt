To install the plugin in Google Chrome and Microsoft Edge browsers, follow these steps:

For Google Chrome:
Open Google Chrome and navigate to chrome://extensions/ in the address bar.
Enable the "Developer mode" toggle in the top right corner.
Click on the "Load unpacked" button on the top left corner.
Browse to the folder where you have the plugin files (manifest.json, background.js, popup.html, popup.js).
Select the folder and click "Select Folder" to install the plugin.
The plugin should now appear in the list of installed extensions.

For Microsoft Edge:
Open Microsoft Edge and navigate to edge://extensions/ in the address bar.
Enable the "Developer mode" toggle in the bottom left corner.
Click on the "Load unpacked" button on the top left corner.
Browse to the folder where you have the plugin files (manifest.json, background.js, popup.html, popup.js).
Select the folder and click "Select Folder" to install the plugin.
The plugin should now appear in the list of installed extensions.
